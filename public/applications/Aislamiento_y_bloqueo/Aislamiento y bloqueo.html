Aislamiento y bloqueo

Bienvenido al procedimiento interactivo de "Aislamiento y bloqueo"

Utilice los controles del mouse para interactuar con el escenario.

  1. Para rotar y desplazarse por las vistas del laboratorio utilice el botón izquierdo del mouse.
  3. Para acercar o alejar las vistas del laboratorio utilice la rueda del mouse.

Bienvenido al procedimiento interactivo de "Trabajos en caliente"

Utilice sus dedos para interactuar con el escenario.

  1. Presione con un dedo para desplazarse en la direcciones indicadas en las flechas ← ↑ → ↓, así como también para seleccionar los botones relacionados a cada paso del procedimiento.
  3. Utilice el gesto de pellizco con sus dedos para acercar o alejar la vista en el módulo interactivo.
  5. Utilice dos dedos y desplácelos hacia arriba o abajo para visualizar el objeto desde la perspectiva deseada.

A continuación, algunas recomendaciones de seguridad para los trabajos de aislamiento y bloqueo.

Identifique que el componente APRON FEEDER se encuentre en estado activo.

Presione los botones rojos de cada tablero resaltado en color celeste para desconectar el dispositivo de control del APRON FEEDER y ELECTROIMAM

Observe la luz que se enciende en el dispositivo de control del APRON FEEDER

Presione sobre la opción "Candados" del checklist para tomar dichos elementos.

Presione sobre la opción "Llaves" del checklist para tomar dichos elementos.

Presione sobre la opción "Pinzas" del checklist para tomar dichos elementos.

Presione sobre la ropa de protección para colocarse dicho elemento.

Presione el casco para colocarse dicho elemento.

Presione sobre los guantes de goma para colocarse dicho elemento.

Presione sobre los guantes de cuero para colocarse dicho elemento.

Presione sobre las botas para colocarse dicho elemento.

Con todos los elementos de protección colocados, continue con los pasos de LockOut y Tag Out.

Presione sobre el elemento resaltado para bajar la palanca del APRON FEEDER

Ahora coloque los elementos de bloqueo. Presione sobre elemento "Pinza" del checklist para colocar dicho elemento.

Ahora coloque los elementos de bloqueo. Presione sobre elemento "Candado" del checklist para colocar dicho elemento.

Ahora coloque los elementos de bloqueo. Presione sobre elemento "Tarjeta" del checklist para colocar dicho elemento.

Ahora, presione sobre el elemento resaltado para bajar la palanca del ELECTROIMAN

Ahora coloque los elementos de bloqueo. Presione sobre elemento "Pinza" del checklist para colocar dicho elemento.

Ahora coloque los elementos de bloqueo. Presione sobre elemento "Candado" del checklist para colocar dicho elemento.

Ahora coloque los elementos de bloqueo. Presione sobre elemento "Tarjeta" del checklist para colocar dicho elemento.

Presione sobre la tapa para abrir la caja de bloqueo.

Presione sobre el botón de guardar las llaves para almacenarlas en la caja de bloqueo.

Ahora, se debe colocar el candado y tarjeta grupal.

Presione sobre la llave del candado grupal resaltada para guardarla.

Presione los botones verdes de los tableros para validar el proceso de aislamiento del APRON FEEDER y ELECTROIMÁN.

Observe que no se enciende ninguna luz, lo que indica que el sistema del APRON FEEDER y ELECTROIMÁN ha sido aislado correctamente.

Finalmente, no se olvide de colocar su tarjeta personal y guardar su llave personal.

¡Felicitaciones! Usted ha culminado el Procedimiento de aislación, bloqueo y etiquetado con éxito.
